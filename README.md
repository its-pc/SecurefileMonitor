# 🔐 Secure File Integrity Checker (GUI + Real-Time)

## ✅ Features
- GUI using Tkinter
- Real-time monitoring with `watchdog`
- SHA-256 hashing for secure file validation
- Alerts on file addition, deletion, or modification
- Beginner-friendly

## 🛠 Requirements
- Python 3.x
- tkinter
- watchdog
